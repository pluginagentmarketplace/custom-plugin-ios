### Issue 2: "I'm overwhelmed by options"

**Solution**:
```
1. Run: /compare-paths [top 3 interests]
2. See clear comparison
3. Pick based on:
   - Interest level
   - Salary potential
   - Job market
   - Learning timeline
4. Commit to 3 months
```

### Issue 3: "Skills aren't loading"

**Solution**:
```
1. Check plugin is installed correctly
2. Reload Claude Code
3. Try explicit skill command:
   /activate skill ai-foundations
4. Check plugin.json references all skills
```

### Issue 4: "Want to switch paths mid-way"

**Solution**:
```
1. Run: /assess current
2. Run: /assess target
3. Run: /compare-paths current vs target
4. Talk to agent about transition
5. Build bridge projects
6. Switch with confidence
```
