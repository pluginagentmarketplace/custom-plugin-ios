### System Design Interview Framework

**5-Step Approach**:
```
1. Clarify Requirements (5 min)
   - Functional requirements
   - Non-functional requirements
   - Constraints (QPS, latency, storage)

2. High-Level Design (10 min)
   - Component overview
   - Data flow diagram
   - Technology choices

3. Deep Dive (10-15 min)
   - Database design
   - Caching strategy
   - API design

4. Bottleneck Analysis (5 min)
   - Scaling database
   - Scaling compute
   - Scaling storage

5. Follow-up (5 min)
   - Monitoring & logging
   - Recovery & failover
   - Cost optimization
```
