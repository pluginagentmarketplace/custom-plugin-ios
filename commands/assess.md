# /assess

Evaluate your knowledge level and get personalized learning recommendations.

This command helps you understand where you stand and what to learn next.

## Usage

```
/assess
/assess [topic]
```

## What This Command Does

1. **Knowledge Assessment**: Answer questions about your experience
2. **Skill Evaluation**: Identify strengths and gaps
3. **Level Determination**: Beginner, Intermediate, or Advanced
4. **Gap Analysis**: See what you need to learn
5. **Recommendations**: Get personalized next steps

## Self-Assessment Examples

### AI/ML Knowledge Assessment
Questions about:
- Python programming experience
- Mathematical background (Linear algebra, calculus, probability)
- ML fundamentals understanding
- Hands-on project experience
- Deployment and production knowledge

**Outcomes**:
- Beginner (0-1 year): Start with AI Foundations
- Intermediate (1-3 years): Focus on production ML
- Advanced (3+ years): Move to MLOps or specialization

### Frontend Skills Assessment
Questions about:
- HTML/CSS/JavaScript mastery
- Framework experience (React, Vue, Angular)
- Testing knowledge
- Performance optimization
- Accessibility understanding

**Outcomes**:
- Beginner: Master fundamentals
- Intermediate: Learn advanced patterns
- Advanced: System design, architecture

### Backend Development Assessment
Questions about:
- Programming language proficiency
- Database knowledge
- API design understanding
- System design exposure
- Production deployment experience

**Outcomes**:
- Beginner: Choose language + master fundamentals
- Intermediate: Add second language + databases
- Advanced: System design + scaling

### DevOps Readiness Assessment
Questions about:
- Linux system administration
- Container knowledge (Docker)
- Cloud platform experience
- CI/CD pipeline familiarity
- Infrastructure as code (Terraform)

**Outcomes**:
- Beginner: Start with Docker
- Intermediate: Add Kubernetes + Terraform
- Advanced: Complex deployment patterns

## Career Level Assessment

Determines your career stage:
- **Junior**: 0-2 years, learning foundations
- **Mid-level**: 2-5 years, expert in specialization
- **Senior**: 5-8 years, architectural thinking
- **Staff/Principal**: 8+ years, strategic impact

## Honest Self-Reflection Questions

Think about:
1. Can I explain this concept to someone else?
2. Have I built projects using this technology?
3. Could I debug complex issues?
4. Do I understand the "why" behind decisions?
5. Can I mentor others in this area?

**If you answer "no" to most, you may not have truly mastered it yet.**

## Learning Recommendations

Based on assessment, you'll get:
- **Primary Focus**: Highest impact area to learn
- **Secondary Skills**: Supporting knowledge
- **Timeline**: Realistic timeframe
- **Resources**: Specific roadmaps and skills
- **Projects**: Portfolio-building recommendations

## Next Steps

After assessment:
1. **Accept recommendations** or customize your path
2. **Use `/learn`** to create detailed learning plan
3. **Follow the recommended skills** from specific agents
4. **Build projects** to reinforce learning
5. **Reassess in 3 months** to track progress
